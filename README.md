# BlueElements
~~~~~~~~~~~~~~~

Description
-----------
This project is a unique blend of PowerPoint and Excel, akin to Access but with extended functionalities such as layout export. What sets it apart is its proprietary database format and script language. The primary goal is to provide a table control capable of handling various data sources seamlessly.

Features
--------
•	Blend of PowerPoint and Excel functionalities
•	Extended features similar to Access
•	Layout export functionality
•	Proprietary database format
•	Custom script language
•	Versatile table control for handling diverse data sources

Navigate to the project directory where you cloned the repository.
